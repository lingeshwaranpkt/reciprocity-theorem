Understanding the Reciprocity Theorem

What is the Reciprocity Theorem?

The Reciprocity Theorem states that in any linear bilateral network, if a voltage source in one branch produces a certain current in another branch, then moving the voltage source to the second branch will produce the same current in the first branch.

Key Principles:

  • Position Exchange: The voltage source (Vs) and ammeter (A) positions are swapped between Figure 2 and Figure 3
  • Same Current: Despite the position swap, the current measured by the ammeter remains identical in both configurations
  • Linear Network: The theorem applies only to linear, bilateral networks with resistive elements
  • Equivalent Resistance: Both circuits have the same equivalent resistance (Req = 4.167 kÎ©)

You can verify the reciprocity theorem by observing that:

  • For Vs = 10V: Current Is = 2.400 mA in both circuits
  • For Vs = 5V: Current Is = 1.200 mA in both circuits
  • For Vs = 15V: Current Is = 3.600 mA in both circuits
  • For Vs = 20V: Current Is = 4.800 mA in both circuits

Practical Applications:

  • Network Analysis: Simplifies complex circuit calculations
  • Antenna Theory: Transmitting and receiving characteristics are reciprocal
  • Filter Design: Input-output relationships remain symmetric
  • Testing: Allows flexible measurement configurations
